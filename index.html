Backup & Restore
Data sudah online & tersinkron! BAP, ADJ, Komplain, dan data lain kini tersimpan di server (Firebase) dan otomatis sama di semua HP karyawan. Backup di bawah ini sebagai cadangan ekstra (opsional tapi disarankan).
Backup Data
Unduh semua data jadi satu file. Simpan di Google Drive, email, atau WhatsApp diri sendiri.
Restore Data
Pulihkan data dari file backup. Data saat ini akan ditimpa dengan isi backup.
Snapshot Otomatis Harian
Sistem otomatis menyimpan cadangan data sekali sehari (7 hari terakhir). Admin bisa memulihkan data ke kondisi tanggal tertentu jika ada kesalahan.
Tips menjaga data aman:
  • Semua data utama kini tersimpan online (Firebase) dan sinkron antar karyawan.
  • Backup berkala tetap berguna sebagai cadangan ekstra (mis. sebelum hapus banyak data).
  • Kirim file backup ke WhatsApp/email diri sendiri agar tersimpan aman.
  • Akun login & data tersimpan di server, tidak hilang walau ganti HP.
